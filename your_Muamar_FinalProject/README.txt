Install Required Libraries
The following Python libraries are required to run the scripts in this toolkit:

nba_api.stats.endpoints: For accessing NBA statistics.
json: For parsing JSON data.
pandas: For data manipulation and analysis.
time: For handling time-related tasks.
os: For interacting with the operating system.
beautifulsoup: For web scraping tasks.
requests: For making HTTP requests.
sklearn: For machine learning and statistical modeling.

You can install these libraries using pip:
pip install nba_api pandas beautifulsoup4 requests scikit-learn


Usage
Each script in the toolkit serves a unique purpose:

injury.py: Analyze and visualize injury data for NBA players.
linearregression.py: Apply linear regression models to predict basketball game outcomes or player performance.
nbastats.py: Gather and analyze NBA player statistics.
PER.py: Calculate the Player Efficiency Rating (PER) for NBA players.
shotchart.py: Visualize shooting performance of players using shot charts.
Totalplayerstats.py: Aggregate comprehensive statistical data of NBA players.

To run a script, use the following command:
python3 [script_name].py

Features
Injury data analysis and visualization

Comprehensive NBA player statistics collection and analysis
